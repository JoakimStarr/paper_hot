## 知网期刊爬取方法（补充版）

### 原有步骤1-8保持不变

1. 访问知网期刊链接：<<https://navi.cnki.net/knavi/journals/index>>
2. 点击a标签中title=经济与管理科学的按钮
3. 获取到div.result中的内容，将a标签的href和title保存到json文件，格式为{title:href}。这是期刊的名字和链接，下面以此期刊获取。目前先获取第一页数据。
4. 进入到上一步的href里面去
5. 获取到div#"YearIssueTree"的内容。先点击dl#2026\_Year\_Issue按钮选择年份，年份限制为{2025,2026}。默认是当年最晚的一期。

7. 然后再点击span#larrow按钮，去到前一期。重复第六步。直到span#larrow的class为btn-larrow disable的时候。停止。并点击获取上一年的数据，如2025。然后重复6、7步。直到获取到所有选择到的年份论文数据链接
8. 获取到所有论文数据后，进入到第六步获取到的href链接里面去，获取论文详细信息

### 补充的详细信息提取步骤

1. 在论文详情页中，提取以下信息：
   - **标题**：div.doc下面的h1的内容
   - **作者信息**：h3.author的内容
   - **摘要**：span.abstract-text的内容
   - **关键词**：p.keywords的内容（多个关键词通常用分号分隔）
2. **补充：DOI等元数据信息提取**

   根据你提供的HTML结构，需要解析`<div class="row">`下的内容：
   ```
   <div class="row">
       <ul>
           <li class="top-space">
               <span class="rowtit">DOI：</span>
               <p>10.19581/j.cnki.ciejournal.2026.03.007</p>
           </li>
           <li class="top-space">
               <span class="rowtit">专辑：</span>
               <p>经济与管理科学</p>
           </li>
           <li class="top-space">
               <span class="rowtit">专题：</span>
               <p>贸易经济;市场研究与信息</p>
           </li>
           <li class="top-space">
               <span class="rowtit">分类号：</span>
               <p class="clc-code">F724.6</p>
           </li>
           <li class="top-space">
               <span class="rowtit">在线公开时间：</span>
               <p>2026-04-08 21:16（知网平台在线公开时间，不代表文献的发表时间）</p>
           </li>
       </ul>
   </div>
   ```
   **解析方法**：
   - 定位到`div.row > ul`元素
   - 遍历每个`li.top-space`元素
   - 在每个`li`中：
     - 获取`span.rowtit`的文本内容作为字段名（如"DOI："、"专辑："等）
     - 获取`p`标签的文本内容作为字段值
     - 注意：分类号字段的`p`标签有特殊class`clc-code`
     - 对于"在线公开时间"字段，需要提取日期时间部分（如"2026-04-08 21:16"），括号内的说明文字可以单独处理或忽略
3. **数据库保存字段（更新版）**：
   - 期刊信息：期刊名称、期刊链接
   - 论文基本信息：论文标题、作者、摘要、关键词
   - 论文元数据：DOI、专辑、专题、分类号、在线公开时间
   - 链接信息：论文链接

### 注意事项

1. **字段清洗**：提取到的字段值可能需要去除首尾空格和特殊字符
2. **多值字段**：如"专题"字段可能包含多个值用分号分隔，建议在数据库中存储为数组或单独的表
3. **时间格式**：在线公开时间建议转换为标准的datetime格式
4. **错误处理**：某些论文可能缺少部分字段，需要做好空值处理
5. **编码问题**：确保正确处理中文字符编码
