# 经济学期刊爬取方法文档

## 1. 管理世界

### 网站结构

- **主页**: <https://glsj.cbpt.cnki.net/WKB2/WebPublication/wkTextContent.aspx?colType=4&yt=2026&st=01>
- **参数说明**:
  - `yt`: 年份 (如 2026)
  - `st`: 期数 (01-12)
- **数据位置**: `div.zxlist` 下的 `ul.column_contbox_zxlist`

### 爬取步骤

1. 构造 URL: `https://glsj.cbpt.cnki.net/WKB2/WebPublication/wkTextContent.aspx?colType=4&yt={year}&st={issue:02d}`
2. 解析 HTML，获取 `div.zxlist` 中的论文列表
3. 提取每篇论文的：
   - 标题: `h3 a` 的文本
   - 作者: `samp` 标签的文本
   - 摘要: `p` 标签的文本
   - 期号: `span` 标签中的期号信息
   - 论文链接: `h3 a` 的 href 属性

### 示例代码

```python
import requests
from bs4 import BeautifulSoup

def fetch_guanli_shijie(year, issue):
    url = f"https://glsj.cbpt.cnki.net/WKB2/WebPublication/wkTextContent.aspx?colType=4&yt={year}&st={issue:02d}"
    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }
    response = requests.get(url, headers=headers)
    soup = BeautifulSoup(response.content, 'html.parser')
    
    papers = []
    zxlist = soup.find('div', class_='zxlist')
    if zxlist:
        items = zxlist.find_all('li')
        for item in items:
            title_elem = item.find('h3').find('a')
            title = title_elem.text.strip()
            link = title_elem['href']
            
            authors = item.find('samp').text.strip()
            abstract = item.find('p').text.strip()
            
            papers.append({
                'title': title,
                'authors': authors.split(';'),
                'abstract': abstract,
                'url': f"https://glsj.cbpt.cnki.net/WKB2/WebPublication/{link}",
                'issue': f"{year}年{issue:02d}期"
            })
    
    return papers
```

## 2. 经济研究

### 网站结构

- **主页**: <https://erj.ajcass.com/#/index?title=本站首页>
- **API 地址**: <https://api.ajcass.com/api/JournalInfoApi/GetIssueinfoList>
- **请求方法**: POST
- **请求参数**:
  ```json
  {
    "JournalID": "201803050001",
    "curr": 1,
    "issue": 3,
    "limit": 50,
    "year": 2026,
    "keywords": ""
  }
  ```
  - `year`: 年份
  - `issue`: 期数 (1-12)
  - 其他参数保持不变

### 返回数据格式

```json
{
  "code": 200,
  "msg": "Success",
  "data": [
    {
      "id": 122064,
      "title": "坚持用中长期规划指导经济社会发展",
      "enTitle": "Persisting in Guiding Economic and Social Development...",
      "authors": "王蕴，毛科俊，于畅",
      "enAuthors": "WANG Yun, MAO Kejun and YU Chang",
      "year": 2025,
      "volume": 60,
      "issue": 12,
      "yearVolumeIssue": "2025,Vol.60(12):5-0",
      "startPageNum": 5,
      "pageNum": "5-0",
      "hits": 458,
      "downloads": 224
    }
  ]
}
```

### 爬取步骤

1. 构造 POST 请求到 API 接口
2. 解析 JSON 响应数据
3. 提取论文信息

### 示例代码

```python
import requests
import json

def fetch_jingji_yanjiu(year, issue):
    url = "https://api.ajcass.com/api/JournalInfoApi/GetIssueinfoList"
    
    payload = {
        "JournalID": "201803050001",
        "curr": 1,
        "issue": issue,
        "limit": 50,
        "year": year,
        "keywords": ""
    }
    
    headers = {
        'Content-Type': 'application/json',
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }
    
    response = requests.post(url, data=json.dumps(payload), headers=headers)
    data = response.json()
    
    papers = []
    if data['code'] == 200:
        for item in data['data']:
            papers.append({
                'title': item['title'],
                'authors': item['authors'].split('，'),
                'abstract': '',  # 需要单独获取详情
                'url': f"https://erj.ajcass.com/#/article/{item['id']}",
                'issue': f"{item['year']}年{item['issue']:02d}期",
                'doi': '',  # 需要从详情页获取
                'hits': item.get('hits', 0),
                'downloads': item.get('downloads', 0)
            })
    
    return papers
```

## 通用爬取策略

### 1. 反爬虫应对

- 使用合理的 User-Agent
- 设置请求间隔（建议 2-5 秒）
- 使用代理 IP 池
- 处理验证码
- 使用 Session 保持状态

### 2. 动态内容处理

- 使用 Selenium 或 Playwright 处理 JavaScript 渲染的内容
- 检查是否有 API 接口（如经济研究）
- 分析网络请求，找到数据接口

### 3. 数据清洗

- 去除 HTML 标签
- 统一作者格式（中文逗号、分号等）
- 清理空白字符
- 验证数据完整性

### 4. 错误处理

- 网络超时重试（最多 3 次）
- 页面解析失败记录
- 数据库写入失败处理
- 异常日志记录

### 5. 日志记录

- 记录爬取开始/结束时间
- 记录成功/失败数量
- 记录错误信息
- 记录爬取的期刊和期号
